## Приклад 1: Птах та Страус (Проблема поведінки)

**Порушення:**
Клас `Ostrich` наслідується від `Bird`, але перевантажує метод `Fly()`, викидаючи помилку, оскільки страуси не літають.

```csharp
public class Bird {
    public virtual void Fly() { /* логіка польоту */ }
}

public class Ostrich : Bird {
    public override void Fly() => throw new NotSupportedException();
}
